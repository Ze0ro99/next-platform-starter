# PiMetaConnect

## 📱 نظرة عامة
PiMetaConnect هو تطبيق متكامل يجمع بين التواصل الاجتماعي، الميتافيرس، والـ NFTs مدعوم بشبكة Pi Network.

## ✨ المميزات
- 🔐 مصادقة آمنة باستخدام Pi Network
- 💰 محفظة رقمية متكاملة
- 💳 نظام دفع بعملة Pi
- 🎨 سوق NFT
- 🌍 تجربة ميتافيرس تفاعلية

## 🚀 التثبيت

### المتطلبات
- متصفح حديث يدعم JavaScript
- حساب Pi Network

### خطوات التشغيل
1. قم بفتح ملف `index.html` في المتصفح
2. أو ارفع الملفات على خادم ويب

## 📦 محتويات الحزمة
```
PiMetaConnect/
├── index.html          # الصفحة الرئيسية
├── app.js              # منطق التطبيق
├── manifest.json       # ملف البيان
├── app_icon_512x512.png # أيقونة التطبيق
└── README.md           # هذا الملف
```

## 🔧 التكوين

### للاستخدام في بيئة الإنتاج
قم بتغيير الإعداد في `app.js`:
```javascript
await Pi.init({
    version: "2.0",
    sandbox: false  // false للإنتاج، true للاختبار
});
```

## 📝 الاستخدام

### تسجيل الدخول
1. انقر على زر "تسجيل الدخول بحساب Pi"
2. وافق على الأذونات المطلوبة
3. سيتم تسجيل دخولك تلقائياً

### إجراء دفعة
1. أدخل المبلغ بعملة Pi
2. أدخل وصف المعاملة
3. انقر على "إرسال الدفعة"
4. أكمل عملية الدفع في تطبيق Pi

## 🔐 الأمان
- جميع المعاملات تتم عبر Pi SDK الرسمي
- لا يتم تخزين بيانات المستخدم محلياً
- المصادقة الثنائية مدعومة
